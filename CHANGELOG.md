# 📦 История изменений — FIST CLICK

## [2.0.0] — 2026-04-29

### ✨ Крупное обновление

#### Новое в FIST CLICK
- **Локализация** — интерфейс на русском и английском, переключение в Настройках
- **Режим «По мыши»** — кликает по текущей позиции курсора без привязки к координатам
- **Скриптовый режим** — отдельная вкладка: список шагов (координата + задержка + кнопка + тип), порядок меняется, количество повторений задаётся
- **Настройки** — отдельная вкладка:
  - Включение/выключение «Поверх всех окон»
  - Смена языка интерфейса
  - Назначение/сброс горячей клавиши
  - Выбор системного шрифта и размера
  - Выбор цвета фона и акцента
  - Сохранение конфига в `fist_click_config.json`
- **Вкладочный интерфейс** — Споты / По мыши / Скрипт / Настройки
- **Более светлая тема** — фон `#181A20` вместо `#0D0D0D`
- **Цвет акцента** — настраивается через color picker

#### Новое в Установщике
- **Автодобавление папки** — при смене пути в диалоге `/FIST-CLICK` добавляется автоматически
- **Hint** под полем пути объясняет поведение
- Тема установщика обновлена в стиль v2.0
