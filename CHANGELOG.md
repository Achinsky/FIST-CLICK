## [1.0.0] — 2026-04-29

### ✨ Первый релиз

#### Добавлено
- Поддержка до **4 независимых точек кликов** (SPOT 1–4)
- Каждая точка включается/выключается отдельно (ON/OFF)
- **Пипетка** (⊕ PICK) — полноэкранный прицел для выбора координат X/Y
- Выбор **кнопки мыши**: ЛЕВАЯ / ПРАВАЯ для каждой точки
- Тип клика: **одиночный (×1)** или **двойной (×2)** для каждой точки
- **Интервал** между кликами: ч : м : с : мс
- Режим длительности:
  - **БЕСКОНЕЧНО** — кликает до нажатия СТОП
  - **ТАЙМЕР** — автоматическая остановка по истечении времени
- **Горячая клавиша** (по умолчанию F6) — ВКЛ/ВЫКЛ без переключения окон
- Переназначение горячей клавиши через кнопку SET
- **Счётчик кликов** в реальном времени
- Окно всегда **поверх всех окон** (`-topmost`)
- Тёмный интерфейс в стиле терминала (Courier New, #FF3B30)
- Поддержка сборки в `.exe` через PyInstaller (`fist_click.spec`)
